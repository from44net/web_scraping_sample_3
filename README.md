# Webスクレイピングサンプル3

## 概要
- 画像を保存するディレクトリの作成
- src属性の取得
- 画像の取得
- 画像の保存

## 開発環境
- Windows 10 Home
- Python 3.9.4
- VS Code 1.55.1

## 仮想環境の作成
```
py-m venv .venv
```

## 仮想環境の有効化(Windows)
```
.venv\Scripts\activate
```

## 必要なパッケージのインストール
```
python -m pip install -r requirements.txt
```
